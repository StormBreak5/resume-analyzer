# Frontend - Analisador de Vagas

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

O aplicativo estará disponível em `http://localhost:3000`

## Build para Produção

```bash
npm run build
npm start
```

## Tecnologias

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Axios
